About VIP Customer Detector

This interactive web app helps small businesses identify VIP customers based on sales data using Recency, Frequency, and Monetary (RFM) scoring. It’s built with FastAPI for the backend and modern HTML, CSS, and JavaScript for a smooth experience.

✅ The VIP score uses total spend, order frequency, and how recently the customer purchased.

✅ It shows the customer’s name and all the products they’ve bought.

✅ You can extend this to send special offers to VIPs, download VIP reports, or connect it with your CRM system.
